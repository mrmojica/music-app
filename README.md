# Spotify Vibe
This application allows users to search and listen to their favorite music using the Spotify API. The user will find the app to have a very simple interface with a cool design.

## Tools
  - React
  - Redux
  - Noda.js
  - Express
  - Spotify API
  - Youtube Iframe API
